什么是框架

框架是地基，地基打的好，不管业务层有多大的变化，都不会影响到正常的开发.

不用框架，理论上讲是可以的，但是开发成本就变高了，很多集成服务要自己搭建

Android 中哪些可以抽象成为框架的：

  1. 网络模块
  2. 图片缓存模块
  3. 数据库模块
  4. UI 基础

把这些抽出来之后，只要写业务层就好了。
